Help: - Скрипты для контроля отображения текста и изображений в соответствии с языком.
LocaleText.cs - скрипт для компонента Text
LocaleImage.cs - скрипт для компонента Image

Dicts: - Словари ключей, которые прописываются в Help-скриптах, с указанием переводов в виде текста или изображений. Каждый словарь представляет из себя экземпляр Scriptable Object, поэтому, все занесенные данные, сохраняются в самих объектах.

0. В списке файлов проекта жмем ПКМ на пустом месте и создаем необходимый словарь(Create->Language->Images/Texts Dictionary).(Добавленный словарь надо подключить в главном скрипте в LocalizationManager.cs)
1. Выбираем создавшийся словарь, в шапке пишем ключ, переводы которого хотим добавить, на пример "Hello", нажимаем "Add".
2. Раскрываем появившийся элемент и в списке "Texts/Images List", нажимаем "+" или устанавливаем числом жилаемое кол-во переводов.
3. Выбираем желаемый язык и прописываем перевод или указываем необходимый спрайт.
4. Создаем или выбираем необходимый объект для локализации, накидываем соотв.скрипт(Locale-Text/Image-.cs), в поле Text ID или Image ID прописываем ключ, который указали в 1-ом пункте.
5. Всё работает автоматически

General:
LocalizationManager.cs - основной скрипт, который ищет переводы и следит за установленным языком, необходима пустышка на сцене
